# LanguageServer(nvim built-in)

`neovim-0.5` から組み込みの LSP-Client 機能が実装される。

<https://neovim.io/doc/user/lsp.html>

各言語ごとの LanguageServer の自動起動などを担うのが lspconfig。

<https://github.com/neovim/nvim-lspconfig>

* go-to-definition
* find-references
* hover
* completion
* rename
* format
* refactor
